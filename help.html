Guide to Use the Application

- Register if you are a new user, otherwise, go to login.

- Login (default: username: kishnuyadav783@gmail.com, password: 6136) or go to register.

- If the student is not registered, enter the USN and name.

- Now click on "Take Picture". It will take around 100 images of the student for training.

- Now click on "Train Image" which converts images into machine numeric code that helps to recognize the students.

- Now you're ready to take attendance. Just click on "Attendance", where you enter the subject name, and that's it. It will automatically mark the attendance.

- If you want to send the mail, click on "Send Mail" that contains attendance.
